TEMA-1:Que es una pagina web.

Una página web es un documento digital accesible a través de internet, que contiene información como texto, imágenes, videos y otros elementos multimedia. Es la unidad básica de un sitio web y puede ser visitada por cualquier persona con un navegador web.

TEMA-2:Que es HTML

HTML, siglas de HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar para crear páginas web. Define la estructura y el contenido de una página web mediante etiquetas que indican cómo debe mostrarse el texto, imágenes, enlaces y otros elementos. No es un lenguaje de programación en sí mismo, sino un lenguaje de marcado, lo que significa que se utiliza para estructurar el contenido y no para realizar operaciones lógicas.

TEMA-3:Estilo de Texto

Se puede controlar el color del texto utilizando el elemento con el atributo COLOR="xxx", donde "xxx" es el nombre en inglés del color que se desea. Hay que tener presente que algunos no se verán o se verán mal si la máquina no soporta, por lo menos, 256 colores.
